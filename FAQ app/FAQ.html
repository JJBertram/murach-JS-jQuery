What is jQuery?

jQuery is a javascript library of functions you're most likely to need as you develop web sites

Why is jQeury becoming so popular?

Three reasons:

  • It's free.
  • It lets you get more done in less time.
  • All of its functions cross-browser compatible.

Which is harder to learn: jQuery or JavaScript?

For more functions, jQuery is significantly easier to learn and use than JavaScript. But remember: jQuery is JavaScript.
